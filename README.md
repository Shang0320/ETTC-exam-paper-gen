# Google Drive API 檔案選擇器


線上出題Ｖ２
這是一個使用 Google Drive API 和 Streamlit 的應用，讓使用者可以從 Google Drive 資料夾中選擇檔案進行下載與處理。

## 功能
- 使用 Google Drive API 列出資料夾內的檔案
- 使用者可以多選檔案進行下載
- 自動讀取 Excel 檔案並顯示前幾行資料
- 內建支援 Service Account 憑證驗證

## 環境需求
- Python 3.8+
- 必要的依賴列於 `requirements.txt`
